About this blog

The intention of this blog is to be a place for me to share my evolving understandings of software engineering and Buddhism. Posts tagged encoding will be practical walkthroughs of a software engineering problem I have encountered and the steps I took to solve it. Posts tagged impermanence will be practical examples of situations in life where an understanding of impermanance has helped me in some way. Posts tagged understandings will (at least for now) just include all posts.

About me

About how this blog is made/hosted

This blog is made with the static site generator Zola which is written in Rust. It is hosted using github pages and published using a github action. The theme is zola-sam a zola port of the Sam theme for Hugo.
